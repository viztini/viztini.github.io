# viztini's tech blog

A retro-styled personal tech blog with Windows 95/98 aesthetics and cyberpunk elements.

## About

This is a static HTML/CSS website exploring low-level programming, OS development, Linux, CLI applications, and AI.

## Features

- Pure HTML and CSS (no frameworks or build tools)
- Windows 95/98 retro aesthetic
- Terminal-style effects and animations
- Fully responsive design
- GitHub Pages compatible

## Deployment

The site is automatically deployed to GitHub Pages via GitHub Actions whenever changes are pushed to the main branch.

## Local Development

Simply open `index.html` in your browser. No build process required!
